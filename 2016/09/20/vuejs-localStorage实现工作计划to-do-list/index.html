vue.js 1.0.xx 语法

vue.js 2.0.xx 语法 废弃的$remove
用 Array.prototype.splice 代替，例如：

methods: {
removeTodo: function (todo) {
var index = this.todos.indexOf(todo)
this.todos.splice(index, 1)
}
}
或者更好的方法，直接给除去的方法一个index参数：

methods: {
removeTodo: function (index) {
this.todos.splice(index, 1)
}
}
